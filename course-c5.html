COURSE 05

노션 & 슬랙
업무생산성 200% UP

4시간 통합 커리큘럼 — 이론 + 실습 병행. 제조 현장 맞춤 시나리오로 노션과 슬랙을 당일 실무에 바로 적용합니다.

⏱ 총 4시간 📋 노션 2시간 💬 슬랙 2시간 🏭 제조업 맞춤
PART01
노션 구조 & 기본 명령어 마스터
핵심 질문 — "지금 우리 업무 소통에서 가장 불편한 게 뭔가요?" 엑셀 버전 혼란 / 카카오톡 공지 묻힘 / 이메일 느림 — 노션이 이 문제를 어떻게 해결하는지 공감부터 시작.
40분
노션 3계층 구조
워크스페이스 · 페이지 · 블록
  • 워크스페이스 — 회사 전체 공간
  • 페이지 — 정보를 담는 단위 (중첩 가능)
  • 블록 — 텍스트·할일·표·이미지 등 최소 단위
  • 실습: 구글 계정으로 로그인 + 앱 설치
슬래시(/) 명령어 & 단축키
빈 줄에서 / → 블록 삽입
  • /h1 대제목 · /h2 중제목 · /h3 소제목
  • /todo 체크박스 · /table 표 · /callout 콜아웃
  • Ctrl+B 굵게 · Ctrl+K 링크 · Ctrl+D 복제
  • 실습: 제목→체크리스트→콜아웃 블록 만들기
PART02
페이지 만들기 & 업무 DB 관리
목표 — 납품일정·불량 현황·회의록을 표·칸반·캘린더 뷰로 자유롭게 관리한다.
60분
작업 보고 페이지 제작 · 25분
아이콘·커버·2단 컬럼 구성
  • 새 페이지 → 아이콘·커버 설정
  • /h1 제목 · /todo 작업 현황 3개 이상
  • /callout 특이사항 또는 긴급 내용
  • 블록 드래그 → 2단 컬럼 (생산현황/품질이슈)
납품일정 DB · 35분
4가지 뷰로 업무 관리
  • DB 속성: 납품명·고객사·담당자·상태·납기일·수량
  • 테이블 뷰 — 전체 데이터 한눈에
  • 칸반 뷰 — 드래그로 상태 변경
  • 캘린더 뷰 — 납기일 기준 월별 표시
📋
▶ 결과물
작업 보고 페이지 + 납품일정 DB (칸반·캘린더 뷰) + 회의록 템플릿 버튼
PART03
슬랙 시작하기 & 채널·메시지
핵심 — 카카오톡·이메일·전화의 문제를 슬랙이 어떻게 해결하는지 이해하고, 채널과 메시지 기능을 익힌다.
50분
슬랙 기초 · 20분
채널 구조 & 화면 구성
  • 공개 채널 (#) — 누구나 참여, 이전 대화 확인
  • 비공개 채널 (🔒) — 초대된 멤버만, 기밀 대화
  • 사이드바·중앙 대화창·상단 아이콘 구성
  • 실습: 워크스페이스 생성 + 앱 설치
메시지 기능 · 30분
멘션·스레드·이모지 반응
  • @멘션 — 특정인 호출 (@channel은 긴급시만)
  • 스레드 — 주제별 답글로 대화 정리
  • 이모지 반응 — ✅ 하나로 읽음·완료 표시
  • DM & 허들 — 1:1 채팅 + 즉석 음성 통화
💬
▶ 결과물
#생산팀 채널 생성 + 멘션·스레드·이모지 반응 실습 완료
PART04
알림 & 검색 & 실무 시나리오
목표 — 금형 라인 긴급 불량 발생 → 슬랙으로 10분 내 해결하는 전체 흐름을 직접 실습한다.
70분
알림 & 검색 · 25분
퇴근 후 방해금지 & 빠른 검색
  • 방해금지 모드 — 퇴근 후 자동 예약
  • 키워드 알림 — "납기", "불량", "긴급" 설정
  • Ctrl+K — 채널·사람 빠른 전환
  • 검색 필터 — in:#채널 / from:이름 / has:file
실무 시나리오 · 45분
금형 불량 발생 → 10분 내 해결
  • STEP 1 — #품질관리 채널에 불량 메시지 + @팀장 멘션
  • STEP 2 — 불량부품 사진 첨부 + 스레드 답글
  • STEP 3 — 허들 음성 통화 + DM 대책 요약
  • STEP 4 — ✅ 이모지 완료 처리 + 검색 확인
🏭
▶ 결과물
#품질관리 채널 실무 대응 흐름 완성 + 채널·스레드·허들·검색 전 기능 활용
