COURSE 08

AI 마케팅 콘텐츠 제작
실무 통합 24시간 과정

Claude + Freepik + Higgsfield + Adobe + CapCut — Adobe에 익숙한 실무자가 AI를 기존 워크플로우에 완전 통합하는 24시간 중급 실무 과정.

24시간 (4H×6회) 중급자 Adobe 숙련자 5개 툴 통합 MCP 자동화
PART01
Claude 실무 자동화 마스터
목표 — Claude 스킬·프로젝트·MCP 연동으로 반복 업무를 자동화하는 시스템을 직접 구축한다.
4시간
PART 1 · 2H
Claude 스킬 & 프로젝트
  • Claude Projects — 팀별 컨텍스트·브랜드 톤 저장
  • 스킬(Skill) 만들기 — 반복 업무용 프롬프트 템플릿
  • 영상 씬 기획 자동화 — 제품 정보 → 콘티·스크립트 생성
  • 광고 문구 A/B 시안 10종 빠르게 생성하는 구조
PART 2 · 2H
MCP 연동 자동화
  • Google Drive MCP — 기획안·결과물 자동 저장
  • Notion MCP — 기획안을 노션 페이지로 자동 생성
  • Higgsfield MCP — Claude에서 영상 프롬프트 직접 전송
  • 제품 정보 → 기획안 → Drive 저장 자동 흐름 구축
결과물
팀 전용 Claude Project / 반복 업무용 스킬 3종 / MCP 자동화 흐름 1개
PART02
Freepik AI — 고퀄리티 마케팅 이미지
목표 — 제품 사진 1장으로 콘셉트 5종 이상 제작, Adobe 통합 후보정까지 완성한다.
4시간
PART 1 · 2H
AI 이미지 생성 전략
  • 제품 사진 1장 → 조명·배경·분위기별 콘셉트 5종+
  • 브랜드 모델·인물 일관성 유지 기법
  • 저화질 이미지 → 인쇄용 고해상도 업스케일
  • Claude로 이미지 프롬프트 생성 → Freepik 바로 적용
PART 2 · 2H
Adobe 통합 후보정
  • AI 이미지 → PSD 레이어로 불러와 합성·리터칭
  • 사진을 일러스트로 변환 / AI 이미지 → 벡터 추출
  • 카탈로그 페이지별 레이아웃 배치
  • Claude AI 검수 — 오탈자·문맥·브랜드 톤 자동 검수
결과물
제품 이미지 5종+ / PSD 합성 완성본 / 벡터 .ai 파일 / 카탈로그 페이지 시안
PART03
Higgsfield — AI 영상 생성 전략
목표 — Kling·Seedance·Cinema Studio 세 모델을 활용해 제품·브랜드·광고 영상을 완성한다.
4시간
PART 1 · 2H
영상 생성 프롬프트 마스터
  • Kling 3.0 — 제품 클로즈업·물리 모션·이미지→영상
  • Seedance 2.0 — 인물 일관성 유지 감성 브랜드 영상
  • Cinema Studio — 달리샷·트래킹·크레인 카메라 연출
  • 조명·질감·색감·카메라 무빙 프롬프트 구조
PART 2 · 2H
광고 영상 자동화
  • Claude → Higgsfield MCP — 기획안 → 영상 자동 생성
  • Marketing Studio — UGC·리뷰·언박싱 광고 자동 생성
  • Freepik 이미지 → Higgsfield 영상으로 확장
  • 편집용 B-roll 클립 세트 제작 전략
결과물
Kling 제품 영상 3종 / Seedance 브랜드 영상 / Cinema Studio 광고 클립 / B-roll 세트
PART04
Adobe Firefly 통합 워크플로우
목표 — 쓰던 Photoshop·Illustrator·After Effects에 AI를 바로 통합한다.
4시간
PART 1 · 2H
Photoshop & Illustrator
  • 생성형 채우기 — 배경 확장·오브젝트 교체·제거
  • Freepik 이미지 + Photoshop Firefly 병용 전략
  • Illustrator 텍스트→벡터 생성·패턴·아이콘 자동 제작
  • AI 생성 이미지 상업적 사용 검수 포인트
PART 2 · 2H
After Effects & Premiere
  • Higgsfield 클립 → Ae 타임라인 워크플로우
  • 색보정·글로우·파티클로 AI 영상 퀄리티 업그레이드
  • Premiere Firefly — 장면 전환·B-roll 자동 삽입
  • AI 영상 어색한 부분 Ae 효과로 보완하는 기법
결과물
Firefly 합성 이미지 / 벡터 아이콘 세트 / Ae 후보정 영상 / Premiere 편집 완성본
PART05
CapCut PC — 숏폼 완성 편집
목표 — 릴스·틱톡·유튜브 쇼츠 멀티채널 규격 영상을 완성한다.
4시간
PART 1 · 2H
AI 기능 고급 활용
  • Higgsfield 클립 → CapCut PC 타임라인 편집
  • AI 자동 자막 생성 + 화자 분리 + 배경음 제거
  • BGM·효과음 AI 매칭 + 화면 전환·템포 조절
PART 2 · 2H
숏폼 구조 & 멀티채널
  • 숏폼 황금 구조 — 후킹 3초·본론·CTA 마무리
  • 15초 / 30초 숏폼 편집 리듬 차이 실습
  • 릴스·틱톡·유튜브 쇼츠 채널별 규격 동시 내보내기
  • Claude로 CTA 문구 생성 → CapCut 자막 바로 적용
결과물
15초 숏폼 완성본 / 30초 홍보 영상 / 멀티채널 규격 영상 3종
PART06
팀별 최종 프로젝트 + MCP 워크플로우 완성
목표 — 팀별 실제 업무 기준 최종 결과물을 완성하고, MCP 자동화 파이프라인을 구축한다.
4시간
PART 1 · 2H
Claude MCP 실전 구축
  • Figma MCP — Claude로 디자인 시안 피드백 자동화
  • Claude → Higgsfield MCP — 기획 텍스트 → 영상 자동 발주
  • 팀별 반복 업무 자동화 파이프라인 구축
  • 프롬프트 스킬 팀 공유 방법 + 버전 관리
PART 2 · 2H
팀별 최종 프로젝트 발표
  • 팀별 실제 업무 기준 최종 결과물 제작
  • AI 결과물 검수 체크리스트 완성
  • 프롬프트 템플릿 정리본 완성
  • 팀별 발표 + 피드백 + 향후 적용 계획
최종 결과물
팀별 완성 결과물 / 검수 체크리스트 / 프롬프트 템플릿 정리본 / MCP 파이프라인
