COURSE 04

AI 이미지 & 영상 제작
Google Flow + Vids 완성 과정

이미지 생성(무료) → 영상 클립 → 편집 마감까지 구글 계정 하나로. 전문 편집 프로그램 없이 홍보 영상을 2시간에 완성합니다.

⏱ 2시간 집중 🎬 4단계 파이프라인 💰 이미지 생성 무료 🖥️ Google 계정만 있으면 OK
PART01
기획 & 대본 작성
핵심 — 영상 제작은 "그림"이 아니라 "말"에서 시작합니다. 대본 없이 시작하면 중간에 방향을 잃습니다.
15분
Gemini 활용대본 생성 프롬프트 구조
  • 역할 지정 → 고객사 정보 → 영상 사양 → 출력 형식 순서로 작성
  • 장면별 출력: 시각 묘사 / 내레이션 / 자막 / 사운드 4가지 항상 포함
  • 시각 묘사 → Google Flow 이미지 프롬프트로 바로 사용
  • 내레이션 → Vids 음성 합성 또는 녹음 MP3 입력
  • 자막 → Vids 오버레이 (화면당 10자 이내 권장)
  • BGM·효과음 → Vids에서 삽입
PART02
이미지 제작 (Google Flow)
핵심 — 좋은 이미지가 있어야 다음 단계 영상 품질이 올라갑니다. 장면당 3~5회 시도 후 최종 1장 선택.
35분
사용 도구 & 비용
이미지 생성 도구 비교
  • Nano Banana — 이미지 편집·변형 (무료)
  • ChatGPT Image 2.0 — 이미지 생성 및 편집 (무료)
  • Nano Banana Pro — 전문가급 제어 (유료)
  • Google Flow — 이미지·영상 생성·편집 통합
프롬프트 6요소 템플릿
이미지 프롬프트 작성법
  • [주제] + [장면] + [카메라·렌즈]
  • [조명] + [스타일] + [비율]
  • 5장을 나란히 봤을 때 "한 영상의 조각 같은가" 확인
  • 톤 일관성이 생명 — 색감·조명 통일
🖼️
▶ 결과물
장면별 이미지 5장 완성 (Scene 1~5 톤 일관성 확보)
PART03
영상 클립 생성 (Google Flow · Veo)
핵심 — "홍보용 60초짜리의 5초 조각" 수준으로 기대치를 맞추세요. 그 수준은 충분히 나옵니다. (유료 플랜 필요)
30분
잘 되는 것
AI 영상 생성 강점
  • 자연 풍경·공간의 카메라 이동
  • 기계·장비의 부분 움직임 (회전·왕복)
  • 인물의 단순 동작
아직 어려운 것
현실적 기대치
  • 복잡한 인물 상호작용
  • 글자·숫자·로고의 정확한 유지
  • 10초 이상 일관된 움직임
  • 사후 편집으로 보완 가능
프롬프트 템플릿영상 프롬프트 6요소
  • [장면] + [주체의 움직임] + [카메라 움직임] + [조명·톤] + [길이]
  • 사후 편집: Insert/Remove Objects · Extend Clip · Camera Adjustment
PART04
Google Vids 편집·마감
핵심 — "Google 슬라이드의 영상 버전" — PowerPoint 다루실 줄 알면 Vids도 바로 하실 수 있습니다.
25분
#작업내용팁
1프로젝트 생성vids.google.com → 새 프로젝트—
2에셋 배열Flow에서 다운로드한 클립·이미지를 타임라인에 배치Drive 경유가 매끄러움
3길이 조정각 장면 길이를 대본 기준으로 조정—
4내레이션AI 음성 합성 또는 녹음 MP3 업로드대표 인사말은 실제 녹음 권장
5자막 삽입대본 자막 필드를 그대로 복사 붙여넣기—
6BGM 조정내레이션 대비 볼륨 30% 이하그 이상이면 내레이션이 묻힘
7내보내기MP4, 1080p 권장—
